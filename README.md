# slinky-json-menu

Welcome to the clone of Slinky Menu !

## Global installation

-   Install Yarn if needed

```shell script
npm install -g yarn
```

## Package installation

-   Install packages

```shell script
yarn
```

## Use your data

-   Replace the content of src/data.json with your json data

## Enjoy !

-   And It's done ! Start the application

```shell script
yarn start
```
